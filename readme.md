# athena-cli

Terminal chat UI built with Ink + React 19.

## Setup

```bash
nvm use           # Use Node 20 (see .nvmrc)
npm install
```

## Commands

```bash
npm run build     # Compile TypeScript
npm run dev       # Watch mode
npm start         # Build & run
npm test          # Run tests
npm run lint      # Check formatting & lint
```

## Run directly

```bash
node dist/cli.js
node dist/cli.js --name=Jane
```
